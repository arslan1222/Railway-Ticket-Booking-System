# Railway Ticket Booking System

A comprehensive web based Railway Ticket Booking System built with PHP, MySQL, Bootstrap, and JavaScript. This system allows passengers to book train tickets online, manage bookings, cancel tickets, and provides administrative features for managing trains, routes, and generating reports.

## Features

### User Features
- User Registration & Login System
- Profile Management (Edit profile, change password)
- Search Trains by Origin, Destination, and Date
- Book Tickets with Passenger Details
- Instant E-Ticket Generation
- Cancel Bookings (80% refund policy)
- View Booking History
- Responsive Design for Mobile Devices

### Admin Features
- Admin Dashboard with Statistics
- Manage Trains (Add, Edit, Delete)
- Manage Routes (Add, Edit, Delete)
- Generate Reports (Monthly/Quarterly/Yearly Income)
- Train List Report
- Upload Train Images

## Technology Stack

- **Backend:** PHP 7.4+ / 8.x
- **Database:** MySQL 5.7+
- **Frontend:** HTML5, CSS3, Bootstrap 5
- **JavaScript:** Vanilla JS, jQuery
- **Icons:** Font Awesome 6
- **Server:** Apache/XAMPP/WAMP/LAMP

## Prerequisites

- PHP 7.4 or higher
- MySQL 5.7 or higher
- Web server (Apache/Nginx)
- XAMPP/WAMP/LAMP (for local development)
